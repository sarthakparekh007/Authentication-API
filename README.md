# Authentication-API

Api end points
'/register'
'/login'
'/login/auth/google'
'/users'
'/profile'
'/profile/edit'

For register require field
- name
- email
- phone
- password

For Login
- email
- password

For profile edit
- name
- email
- phone
- password
- bio
- photo
- isPrivate


For Google Auth You need to add your client id and client secret id
